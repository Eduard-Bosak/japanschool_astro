Хирагана и катакана — **ворота в японский язык**. Без них невозможно двигаться
дальше: учебники, приложения, даже меню в ресторане используют эти азбуки.

Хорошая новость: за **2 недели** можно уверенно освоить обе системы, если
подойти к делу системно.

## Структура японской письменности

### Три системы письма

| Система      | Символов | Назначение                             |
| ------------ | -------- | -------------------------------------- |
| **Хирагана** | 46       | Японские слова, грамматика, окончания  |
| **Катакана** | 46       | Заимствования, звукоподражания, акцент |
| **Кандзи**   | 2136+    | Корни слов, семантика                  |

### Почему две азбуки?

**Хирагана (ひらがな)** — мягкие, округлые символы для:

- Японских слов без кандзи: きれい (красивый)
- Грамматических частиц: は、が、を、に
- Окончаний глаголов: 食べ**る** (taberu)
- Фуриганы (подсказки чтения над кандзи)

**Катакана (カタカナ)** — угловатые символы для:

- Иностранных слов: コーヒー (кофе)
- Звукоподражаний: ワンワン (гав-гав)
- Научных терминов
- Эмфазы (выделения)

## План на 2 недели

### Неделя 1: Хирагана

#### День 1-2: Гласные и первые согласные

**Гласные (5 символов):**

| Хирагана | Романизация | Мнемоника                               |
| -------- | ----------- | --------------------------------------- |
| あ       | a           | «А» удивления — рот открыт              |
| い       | i           | Две «I» (палочки) стоят рядом           |
| う       | u           | Буква «U» сверху, закорючка             |
| え       | e           | «Э» — напоминает число 3 с хвостиком    |
| お       | o           | «О» — человек кланяется, круг на голове |

**Ka-ряд (5 символов):**

| Хирагана | Романизация | Мнемоника          |
| -------- | ----------- | ------------------ |
| か       | ka          | Меч рубит — «ka!»  |
| き       | ki          | Ключ (key)         |
| く       | ku          | Клюв птицы         |
| け       | ke          | Кегли для боулинга |
| こ       | ko          | Два червячка       |

**Практика:** Пишите каждый символ 10 раз, произнося вслух.

#### День 3-4: Согласные S, T, N, H

**Sa-ряд:**

- さ (sa) — «Садовник» с лопатой
- し (shi) — Рыболовный крючок «fishing»
- す (su) — «Суп» в ложке
- せ (se) — «Секундная» стрелка
- そ (so) — «Соловей» поёт

**Ta-ряд:**

- た (ta) — «Та самая» буква с хвостом
- ち (chi) — «Cheerleader» с помпонами
- つ (tsu) — «Tsunami» волна
- て (te) — «Телефон» с трубкой
- と (to) — «Торнадо»

**Na-ряд:**

- な (na) — «Узел» (knot)
- に (ni) — «Колени» (knee)
- ぬ (nu) — «Лапша» (noodle)
- ね (ne) — «Гнездо» (nest)
- の (no) — «No» — перечёркнутый круг

**Ha-ряд:**

- は (ha) — «Дом» (house)
- ひ (hi) — Улыбка «хи-хи»
- ふ (fu/hu) — «Фудзи» гора
- へ (he) — Холм
- ほ (ho) — «Хоббит» с посохом

#### День 5-6: M, Y, R, W, N

**Ma-ряд:**

- ま (ma) — «Мама» с двумя глазами
- み (mi) — Цифра 21
- む (mu) — Корова «муууу»
- め (me) — Глаз
- も (mo) — Рыболовный крючок «больше»

**Ya-ряд (3 символа):**

- や (ya) — «Яхта»
- ゆ (yu) — Рыба
- よ (yo) — «Йо-йо» на верёвке

**Ra-ряд:**

- ら (ra) — «Русалка»
- り (ri) — Река (river)
- る (ru) — Петля
- れ (re) — «Дождь» (rain)
- ろ (ro) — «Ро» — число 3

**Wa-ряд и N:**

- わ (wa) — «Вода»
- を (wo) — Только частица
- ん (n) — Единственная согласная без гласной

#### День 7: Озвончение и твёрдые согласные

**Дакутэн (゛) — озвончение:**

- か → が (ka → ga)
- さ → ざ (sa → za)
- た → だ (ta → da)
- は → ば (ha → ba)

**Хандакутэн (゜) — только для H:**

- は → ぱ (ha → pa)
- ひ → ぴ (hi → pi)

**Сочетания с маленькими я/ю/ё:**

- きゃ (kya), きゅ (kyu), きょ (kyo)
- しゃ (sha), しゅ (shu), しょ (sho)
- ちゃ (cha), ちゅ (chu), ちょ (cho)

### Неделя 2: Катакана

#### Параллельное изучение

Катакана повторяет хирагану по звукам, но символы другие:

| Хирагана | Катакана | Звук |
| -------- | -------- | ---- |
| あ       | ア       | a    |
| い       | イ       | i    |
| う       | ウ       | u    |
| え       | エ       | e    |
| お       | オ       | o    |

**Стратегия:** Учите парами — сначала хирагана, сразу катакана.

#### День 8-10: Основные символы

**Гласные:**

- ア (a) — Антенна
- イ (i) — Два «I» под углом
- ウ (u) — Улитка
- エ (e) — «Elevator»
- オ (o) — Оператор

**Ka-ряд:**

- カ (ka) — «Katana» меч
- キ (ki) — «Key» ключ
- ク (ku) — «Cool» угол
- ケ (ke) — «K» с хвостом
- コ (ko) — Угол комнаты

**Продолжайте по аналогии с хирагана**, создавая свои ассоциации.

#### День 11-12: Сочетания и долгие гласные

**Долгие гласные в катакана:**

- ー (chōon) — горизонтальная черта
- コーヒー (kōhī) — кофе
- ケーキ (kēki) — торт
- テーブル (tēburu) — стол

**Удвоение согласных:**

- ッ (маленькое tsu) — пауза
- カップ (kappu) — чашка
- サッカー (sakkā) — футбол

#### День 13-14: Практика чтения

**Читайте катакану везде:**

- Меню кафе: ピザ (pizza), パスタ (pasta)
- Названия брендов: ソニー (Sony), トヨタ (Toyota)
- Вывески магазинов: スーパー (supermarket), コンビニ (convenience store)

## Методы запоминания

### 1. Мнемотехника (истории)

Создавайте яркие, абсурдные образы для каждого символа.

**Плохая мнемоника:** «あ похоже на букву A»

**Хорошая мнемоника:** «あ — это человек, который кричит А-а-а! от удивления,
руки раскинуты, рот открыт»

### 2. Карточки (SRS)

**Структура карточки:**

_Лицевая:_

```
あ
```

_Обратная:_

```
Звук: a
Мнемоника: Человек кричит «А!»
Слово: あめ (ame) — дождь
```

**Приложения:**

- **Anki** — бесплатно, мощное SRS
- **Kana Mind** — специально для азбук
- **Dr. Moku** — с картинками-мнемониками

### 3. Прописи

**Физическое письмо** укрепляет память. Порядок черт важен!

**Правила порядка черт:**

1. Сверху вниз
2. Слева направо
3. Горизонталь перед вертикалью
4. Пересекающая черта — последняя

**Где взять прописи:**

- Книга «Japanese Hiragana & Katakana for Beginners»
- Бесплатные PDF на realkana.com
- Приложение «Kana — Hiragana and Katakana»

### 4. Чтение с первого дня

**Не ждите, пока выучите всё!** Читайте то, что знаете:

_День 1:_ あ、い、う — уже можете прочитать あい (ai — любовь)

_День 3:_ Добавились か、き — читайте かき (kaki — хурма)

_День 5:_ Уже целые слова: さかな (sakana — рыба), きたない (kitanai — грязный)

## Практические упражнения

### Упражнение 1: Диктант

Попросите кого-то диктовать слоги, записывайте:

1. a, i, u, e, o
2. ka, ki, ku, ke, ko
3. Смешанные: u, ko, a, ke, i

### Упражнение 2: Скороговорки

**Хирагана:**

- なまむぎ なまごめ なまたまご (Namymugi namagome namatamago — Сырая пшеница,
  сырой рис, сырое яйцо)

**Катакана:**

- ブラジル人の ミラさん (Burajirujin no Mira-san — Мира из Бразилии)

### Упражнение 3: Поиск катакана

Откройте любой японский сайт (например, amazon.co.jp) и найдите 10 слов
катакана. Попробуйте их прочитать:

- カメラ (kamera) — камера
- パソコン (pasokon) — персональный компьютер
- インターネット (intānetto) — интернет

### Упражнение 4: Транскрипция имён

Напишите катакана:

- Ваше имя
- Имена друзей
- Название города

**Пример:**

- Анна → アンナ (Anna)
- Москва → モスクワ (Mosukuwa)
- Кофе → コーヒー (Kōhī)

## Распространённые ошибки

### ❌ Учить только чтение без письма

**Проблема:** Путаете похожие символы.

**Решение:** Пишите каждый символ минимум 20 раз.

### ❌ Пропускать катакана

**Проблема:** Не можете читать заимствования, которых в современном японском
очень много.

**Решение:** Катакана так же важна, как хирагана. Учите параллельно.

### ❌ Игнорировать озвончение

**Проблема:** Не отличаете か (ka) от が (ga).

**Решение:** Практикуйте пары: ka-ga, sa-za, ta-da, ha-ba-pa.

### ❌ Учить символы без слов

**Проблема:** Знаете символы, но не узнаёте их в словах.

**Решение:** Каждый символ — со словом. あ → あめ (дождь), い → いぬ (собака).
