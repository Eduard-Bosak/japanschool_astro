Собираетесь в Японию? Эти фразы **спасут вас в любой ситуации**, даже если вы
никогда не учили японский.

## Базовые правила вежливости

### Волшебные слова

| Японский             | Ромадзи            | Когда использовать            |
| -------------------- | ------------------ | ----------------------------- |
| ありがとうございます | Arigatou gozaimasu | Спасибо (вежливо)             |
| すみません           | Sumimasen          | Извините / Простите           |
| ごめんなさい         | Gomennasai         | Извините (более личное)       |
| お願いします         | Onegaishimasu      | Пожалуйста (просьба)          |
| はい / いいえ        | Hai / Iie          | Да / Нет                      |
| どういたしまして     | Dou itashimashite  | Пожалуйста (ответ на спасибо) |

> 💡 **Супер-фраза:** すみません (sumimasen) работает везде! Привлечь внимание,
> извиниться, попросить помощь.

## В аэропорту

### Прибытие

```
荷物受取所はどこですか？
Nimotsu uketorijo wa doko desu ka?
Где получение багажа?

トイレはどこですか？
Toire wa doko desu ka?
Где туалет?

駅への行き方を教えてください。
Eki e no ikikata wo oshiete kudasai.
Подскажите как добраться до станции.

タクシー乗り場はどこですか？
Takushii noriba wa doko desu ka?
Где стоянка такси?
```

### Полезные слова

- パスポート (pasupooto) — паспорт
- 入国審査 (nyuukoku shinsa) — иммиграционный контроль
- 税関 (zeikan) — таможня
- 両替 (ryougae) — обмен валюты

## В отеле / хостеле

### Заселение

```
予約しているエドゥアルドです。
Yoyaku shite iru Edoarudo desu.
У меня бронь на имя Эдуард.

チェックインお願いします。
Checkuin onegaishimasu.
Регистрация, пожалуйста.

Wi-Fiのパスワードは何ですか?
Wi-Fi no pasuwaado wa nan desu ka?
Какой пароль от Wi-Fi?

朝食は何時からですか?
Choushoku wa nanji kara desu ka?
Во сколько завтрак?
```

### Проблемы

```
部屋が寒いです。
Heya ga samui desu.
В комнате холодно.

お湯が出ません。
Oyu ga demasen.
Нет горячей воды.

タオルをもう一枚お願いします。
Taoru wo mou ichimai onegaishimasu.
Ещё одно полотенце, пожалуйста.
```

### Выезд

```
チェックアウトお願いします。
Checkuauto onegaishimasu.
Выселение, пожалуйста.

荷物を預かってもらえますか?
Nimotsu wo azukatte moraemasu ka?
Можно оставить багаж?
```

## В ресторане / кафе

### Вход и заказ

```
2名です。
Futari desu.
Нас двое.

メニューをお願いします。
Menyuu wo onegaishimasu.
Меню, пожалуйста.

これをください。
Kore wo kudasai.
Это, пожалуйста. (указываете на картинку/блюдо)

お水をください。
Omizu wo kudasai.
Воды, пожалуйста.

おすすめは何ですか?
Osusume wa nan desu ka?
Что рекомендуете?
```

### Диетические ограничения

```
ベジタリアンです。
Bejitarian desu.
Я вегетарианец.

アレルギーがあります。
Arerugii ga arimasu.
У меня аллергия.

これに肉が入っていますか?
Kore ni niku ga haitte imasu ka?
Здесь есть мясо?

辛くないものはありますか?
Karakunai mono wa arimasu ka?
Есть что-то неострое?
```

### Завершение

```
お会計お願いします。
Okaikei onegaishimasu.
Счёт, пожалуйста.

ごちそうさまでした。
Gochisousama deshita.
Спасибо за еду (при выходе).

別々に払えますか?
Betsubetsu ni haraemasu ka?
Можно оплатить раздельно?

カードで払えますか?
Kaado de haraemasu ka?
Можно картой?
```

> 💡 В Японии не принято давать чаевые! Это может обидеть.

## Транспорт

### Поезда и метро

```
渋谷まではいくらですか?
Shibuya made wa ikura desu ka?
Сколько стоит до Сибуи?

切符売り場はどこですか?
Kippu uriba wa doko desu ka?
Где продаются билеты?

新宿へ行きたいです。
Shinjuku e ikitai desu.
Хочу поехать в Синдзюку.

何番線ですか?
Nanban-sen desu ka?
Какая платформа?

次の駅は何ですか?
Tsugi no eki wa nan desu ka?
Какая следующая станция?
```

### Такси

```
〇〇ホテルまでお願いします。
[Hotel name] made onegaishimasu.
До отеля [название], пожалуйста.

ここで止めてください。
Koko de tomete kudasai.
Остановите здесь.

いくらですか?
Ikura desu ka?
Сколько?
```

## Покупки

### В магазине

```
これを見せてください。
Kore wo misete kudasai.
Покажите это, пожалуйста.

試着できますか?
Shichaku dekimasu ka?
Можно примерить?

もっと大きいサイズはありますか?
Motto ookii saizu wa arimasu ka?
Есть размер побольше?

他の色はありますか?
Hoka no iro wa arimasu ka?
Есть другие цвета?

いくらですか?
Ikura desu ka?
Сколько стоит?

高すぎます。
Takasugimasu.
Слишком дорого.

安くなりませんか?
Yasuku narimasen ka?
Дешевле не будет? (торг)
```

### Конбини (круглосуточный магазин)

```
袋をください。
Fukuro wo kudasai.
Пакет, пожалуйста.

レシートは要りません。
Reshiito wa irimasen.
Чек не нужен.

これを温めてください。
Kore wo atatamete kudasai.
Подогрейте это, пожалуйста.
```

## Спрашивать дорогу

```
道に迷いました。
Michi ni mayoimashita.
Я заблудился.

〇〇への行き方を教えてください。
[Place] e no ikikata wo oshiete kudasai.
Как пройти к [место]?

ここはどこですか?
Koko wa doko desu ka?
Где я нахожусь?

まっすぐ行ってください。
Massugu itte kudasai.
Идите прямо.

右/左に曲がってください。
Migi/Hidari ni magatte kudasai.
Поверните направо/налево.

駅はどこですか?
Eki wa doko desu ka?
Где станция?

近いですか?
Chikai desu ka?
Это близко?

歩いて行けますか?
Aruite ikemasu ka?
Можно дойти пешком?
```

## Экстренные ситуации

### Помощь

```
助けてください！
Tasukete kudasai!
Помогите!

警察を呼んでください！
Keisatsu wo yonde kudasai!
Вызовите полицию!

救急車を呼んでください！
Kyuukyuusha wo yonde kudasai!
Вызовите скорую!

火事です！
Kaji desu!
Пожар!

病院へ連れて行ってください。
Byouin e tsurete itte kudasai.
Отвезите в больницу.
```

### Здоровье

```
気分が悪いです。
Kibun ga warui desu.
Мне плохо.

頭が痛いです。
Atama ga itai desu.
Болит голова.

お腹が痛いです。
Onaka ga itai desu.
Болит живот.

薬局はどこですか?
Yakkyoku wa doko desu ka?
Где аптека?

英語を話せる医者はいますか?
Eigo wo hanaseru isha wa imasu ka?
Есть врач, говорящий по-английски?
```

### Потери

```
財布を無くしました。
Saifu wo nakushimashita.
Я потерял кошелёк.

パスポートを盗まれました。
Pasupooto wo nusumaremashita.
У меня украли паспорт.

大使館への行き方を教えてください。
Taishikan e no ikikata wo oshiete kudasai.
Как пройти в посольство?
```

## Полезные вопросы

```
英語を話せますか?
Eigo wo hanasemasu ka?
Вы говорите по-английски?

もう一度お願いします。
Mou ichido onegaishimasu.
Повторите, пожалуйста.

ゆっくり話してください。
Yukkuri hanashite kudasai.
Говорите медленнее, пожалуйста.

書いてもらえますか?
Kaite moraemasu ka?
Можете написать?

写真を撮ってもいいですか?
Shashin wo totte mo ii desu ka?
Можно сфотографировать?
```

## Приложения must-have

1. **Google Translate** — camera mode для меню
2. **Google Maps** — офлайн карты
3. **Hyperdia** — расписание поездов
4. **Suica** — электронный кошелёк для транспорта
5. **Tabelog** — рейтинги ресторанов

## Культурные правила

✅ **Кланяйтесь** при встрече ✅ **Снимайте обувь** при входе в дом/храм ✅
**Говорите тихо** в транспорте ✅ **Не едьте/пейте на ходу** ✅ **Не оставляйте
чаевых** ❌ **Не разговаривайте по телефону** в поезде ❌ **Не указывайте
палочками** на людей ❌ **Не втыкайте палочки** вертикально в рис

## Шпаргалка для печати

Распечатайте и носите с собой:

```
すみません (Sumimasen) - Извините
ありがとうございます (Arigatou gozaimasu) - Спасибо
トイレはどこですか? (Toire wa doko desu ka?) - Где туалет?
いくらですか? (Ikura desu ka?) - Сколько стоит?
これをください (Kore wo kudasai) - Это, пожалуйста
助けてください (Tasukete kudasai) - Помогите!
英語を話せますか? (Eigo wo hanasemasu ka?) - Вы говорите по-английски?
```

## Заключение

С этими фразами вы **выживете** в Японии, даже не зная языка.

**Главное правило:** Японцы ценят попытки говорить на их языке. Даже если
произнесёте с акцентом — вам помогут с улыбкой.

良い旅を！(Yoi tabi wo! — Хорошего путешествия!)
