Вы решили учить японский. Отлично! Это **один из самых увлекательных языков**:
уникальная письменность, богатая культура, огромное количество контента.

Но с чего начать? Этот гайд проведёт вас через первые месяцы обучения.

## Реалистичные ожидания

### Сколько времени нужно?

| Уровень | Время (при 10 ч/неделю) | Что сможете                              |
| ------- | ----------------------- | ---------------------------------------- |
| N5      | 3-4 месяца              | Базовые диалоги, простые тексты          |
| N4      | +4-6 месяцев            | Повседневное общение, простые статьи     |
| N3      | +8-12 месяцев           | Рабочее общение, неадаптированные тексты |

**Важно:** Это при **регулярных** занятиях. Учить «когда есть время» займёт в
2-3 раза дольше.

### Японский — действительно сложный?

**Да и нет.**

**Сложное:**

- Три системы письма (хирагана, катакана, кандзи)
- Грамматика отличается от европейских языков
- Уровни вежливости и формальности
- 2136 иероглифов для свободного чтения

**Простое:**

- Произношение легче английского (5 гласных, чёткие звуки)
- Нет артиклей, падежей, рода
- Грамматика логичная, исключений мало
- Письмо фонетическое (хирагана/катакана)

## Этап 1: Азбуки (2-4 недели)

### Хирагана — первая

**46 символов** для японских слов и грамматики.

```
あ い う え お
か き く け こ
さ し す せ そ
た ち つ て と
な に ぬ ね の
は ひ ふ へ ほ
ま み む め も
や   ゆ   よ
ら り る れ ろ
わ       を
ん
```

**Как учить:**

1. По 5-10 символов в день
2. Мнемоники (образы-ассоциации)
3. Прописи — писать вручную
4. Приложения: Kana Mind, Dr. Moku

### Катакана — вторая

**46 символов** для иностранных слов.

```
ア イ ウ エ オ
カ キ ク ケ コ
...
```

**Почему важна:**

- コーヒー (кофе), パソコン (компьютер), インターネット (интернет)
- В современном японском заимствований очень много

### Не начинайте с ромадзи!

**Ромадзи** (латиница) — костыль, который мешает развитию. Сразу читайте каной.

❌ Плохо: «konnichiwa, watashi wa Anna desu»

✅ Хорошо: 「こんにちは、わたしはアンナです」

## Этап 2: Базовая грамматика (1-2 месяца)

### Структура предложения

Японский: **Подлежащее + Дополнение + Сказуемое**

```
私は   りんごを   食べます
Watashi wa  ringo wo   tabemasu
Я      яблоко     ем
```

**Глагол всегда в конце!**

### Ключевые частицы

| Частица | Функция               | Пример                                      |
| ------- | --------------------- | ------------------------------------------- |
| は      | Тема предложения      | 私**は**学生です (Я — студент)              |
| が      | Подлежащее            | 雨**が**降っている (Дождь идёт)             |
| を      | Прямое дополнение     | 本**を**読む (Читать книгу)                 |
| に      | Направление/время     | 学校**に**行く (Идти в школу)               |
| で      | Место действия/способ | 図書館**で**勉強する (Учиться в библиотеке) |
| の      | Притяжание            | 私**の**本 (Моя книга)                      |

### Глагольные формы

**Вежливая форма (-ます):**

- 食べます (tabemasu) — ем
- 飲みます (nomimasu) — пью
- 行きます (ikimasu) — иду

**Прошедшее время:**

- 食べました (tabemashita) — ел
- 飲みました (nomimashita) — пил

**Отрицание:**

- 食べません (tabemasen) — не ем
- 食べませんでした (tabemasen deshita) — не ел

### Первые прилагательные

**い-прилагательные:**

- 大きい (ookii) — большой
- 小さい (chiisai) — маленький
- 新しい (atarashii) — новый

**な-прилагательные:**

- きれい**な** (kirei na) — красивый
- 静か**な** (shizuka na) — тихий
- 元気**な** (genki na) — бодрый

## Этап 3: Первая лексика (параллельно)

### Сколько слов нужно?

| Уровень | Словарный запас |
| ------- | --------------- |
| N5      | 800 слов        |
| N4      | 1500 слов       |
| N3      | 3750 слов       |

### С каких слов начать?

**Категория 1: Личные местоимения**

- 私 (watashi) — я
- あなた (anata) — вы
- 彼/彼女 (kare/kanojo) — он/она

**Категория 2: Базовые глаголы**

- 食べる (taberu) — есть
- 飲む (nomu) — пить
- 行く (iku) — идти
- 来る (kuru) — приходить
- 見る (miru) — смотреть
- 聞く (kiku) — слушать

**Категория 3: Числительные**

- 一、二、三、四、五 (ichi, ni, san, shi/yon, go)
- 六、七、八、九、十 (roku, shichi/nana, hachi, ku/kyuu, juu)

**Категория 4: Время**

- 今日 (kyou) — сегодня
- 明日 (ashita) — завтра
- 昨日 (kinou) — вчера
- 今 (ima) — сейчас

### Как учить слова

1. **Контекст, не списки** — слова в предложениях
2. **Anki/SRS** — интервальное повторение
3. **Аудио** — произношение с первого дня
4. **Кандзи постепенно** — не только хирагана

## Этап 4: Первые кандзи (со 2-3 месяца)

### Не откладывайте!

Многие боятся кандзи и откладывают их. Это ошибка:

❌ «Сначала выучу грамматику, потом кандзи»

✅ «Учу кандзи параллельно с первых недель»

### С каких начать?

**N5 кандзи (100 штук):**

```
日 月 火 水 木 金 土
一 二 三 四 五 六 七 八 九 十
人 女 男 子 学 生 先
```

### Метод изучения

1. **Разбор на радикалы**
   - 休 = 人 (человек) + 木 (дерево) → «Человек отдыхает под деревом»

2. **Учить оба чтения**
   - Он'ёми (китайское): 山 = サン
   - Кун'ёми (японское): 山 = やま

3. **В составе слов**
   - 山 → 山田 (Ямада), 富士山 (Фудзияма)

## Типичные ошибки новичков

### ❌ Ошибка 1: Учить только разговорный

**Проблема:** Не можете читать, не понимаете письменный японский.

**Решение:** С первого дня — азбуки и чтение.

### ❌ Ошибка 2: Переводить буквально

**Проблема:** Японская грамматика работает иначе.

```
Русский: У меня есть книга
Буквально: 私は本を持っている (Я книгу имею)
Естественно: 本がある (Книга существует)
```

**Решение:** Думайте по-японски, не переводите.

### ❌ Ошибка 3: Игнорировать аудирование

**Проблема:** Понимаете текст, но не понимаете речь.

**Решение:** С первого месяца — подкасты, аниме, музыка.

### ❌ Ошибка 4: Учить без системы

**Проблема:** Прыгаете между ресурсами, нет прогресса.

**Решение:** Один основной учебник + дополнительные ресурсы.

### ❌ Ошибка 5: Ждать идеального момента

**Проблема:** «Начну, когда будет больше времени».

**Решение:** Лучше 15 минут сегодня, чем 2 часа «когда-нибудь».

## Рекомендуемые ресурсы

### Учебники

**Для самостоятельного изучения:**

- **Genki I, II** — стандарт для университетов
- **Minna no Nihongo** — классика, полное погружение
- **Japanese from Zero! 1-5** — дружелюбный для новичков
- **Marugoto** — коммуникативный подход

### Приложения

**Азбуки:**

- Kana Mind, Dr. Moku

**Словарный запас:**

- Anki (с колодой Core 2K/6K)
- WaniKani (кандзи)
- Bunpo (грамматика)

**Практика:**

- HelloTalk (разговор с носителями)
- iTalki (репетиторы)
- Tandem (языковой обмен)

### Онлайн-курсы

- **JapanesePod101** — аудио уроки
- **Tae Kim's Guide** — бесплатная грамматика
- **Bunpro.jp** — SRS для грамматики
- **NHK World Easy Japanese** — для начинающих

### YouTube каналы

- **Japanese Ammo with Misa** — грамматика
- **Dogen** — произношение
- **Nihongo no Mori** — уроки на японском
- **Cure Dolly** — логичный подход к грамматике

## Примерное расписание новичка

### Ежедневно (30-60 минут)

| Время  | Активность                      |
| ------ | ------------------------------- |
| 10 мин | Anki (повторение + новые слова) |
| 15 мин | Грамматика (1 тема из учебника) |
| 10 мин | Чтение (простые тексты)         |
| 10 мин | Аудирование (подкаст, видео)    |
| 5 мин  | Прописи (кана или кандзи)       |

### Еженедельно

- **1 урок из учебника** (с упражнениями)
- **Разговорная практика** (даже с собой!)
- **1 эпизод аниме** без субтитров/с японскими
- **Обзор недели** — что выучили, что забыли

## Как оставаться мотивированным

### 1. Найдите своё «зачем»

- Аниме без субтитров?
- Работа в Японии?
- Путешествия?
- Общение с друзьями?

Чёткая цель = топливо для мотивации.

### 2. Отмечайте прогресс

- Ведите дневник изучения
- Записывайте количество выученных слов
- Пересматривайте старые записи — увидите рост

### 3. Погружайтесь в культуру

- Смотрите аниме/дорамы
- Слушайте J-pop/J-rock
- Читайте мангу
- Готовьте японские блюда

### 4. Найдите сообщество

- Разговорные клубы
- Discord серверы
- Языковые партнёры
- Курсы с группой

### 5. Прощайте себе ошибки

Забыли слово? Нормально. Не занимались неделю? Бывает. Главное — продолжать.

## Первые 100 дней

### День 1-14: Азбуки

- Хирагана (все 46 + дакутэн)
- Начать катакану

### День 15-30: Катакана + база

- Катакана (все 46 + комбинации)
- Первые 50 слов
- Структура предложения

### День 31-60: Грамматика

- Частицы は、が、を、に、で
- Глаголы в ます-форме
- Прилагательные
- Первые 200 слов
- Первые 20 кандзи

### День 61-100: Практика

- Простые диалоги
- Чтение детских текстов
- 400+ слов
- 50+ кандзи
- Первый мини-тест N5

## Результат через 100 дней

При регулярной практике (1 час/день) вы сможете:

✅ Читать хирагану и катакану свободно

✅ Понимать простые предложения

✅ Представиться и вести базовый диалог

✅ Читать 50-100 кандзи

✅ Понимать медленную речь

✅ Писать короткие тексты

## Когда идти к преподавателю?

**Сразу** — если хотите:

- Структурированное обучение
- Исправление ошибок с первого дня
- Разговорную практику
- Мотивацию и ответственность

**После базы** — если:

- Любите самостоятельное обучение
- Хотите понять, подходит ли вам японский
- Бюджет ограничен

**Важно:** Даже при самостоятельном обучении периодические консультации с
носителем/преподавателем очень полезны для коррекции курса.

## Заключение

Японский — это марафон, а не спринт. **Не существует «быстрого» способа**
выучить язык с тремя системами письма и 2000+ иероглифов.

Но **каждый день** — это шаг к цели. Через год регулярных занятий вы будете
удивлены своим прогрессом.

始めましょう！(Hajimemashou — Давайте начнём!)
